Molecular viewer — technical specification

v1.0 React 19 · Three.js r168 · TypeScript 5 · Vite 6 Web app · PWA

1. Overview

A browser-based interactive 3D molecular viewer. Users upload PDB or mmCIF files (or fetch by RCSB accession code) and explore the structure using multiple rendering modes: ball-and-stick, space-filling, ribbon, and backbone. The viewer supports atom selection, distance measurement, secondary structure highlighting, and screenshot export. Performance must support proteins up to ~20,000 atoms at 60 fps on a mid-range desktop.

4. Data model

Define in src/parsers/types.ts. All coordinate values in Ångströms.

interface Atom {
  serial: number
  name: string           // e.g. "CA", "N", "O"
  element: string        // e.g. "C", "N", "O", "H"
  residueName: string    // e.g. "ALA", "GLY"
  chainID: string
  residueSeq: number
  x: number
  y: number
  z: number
  bFactor?: number
  occupancy?: number
}

interface Bond {
  atomA: number          // serial index
  atomB: number
  order: 1 | 2 | 3
}

interface Residue {
  seq: number
  name: string
  chainID: string
  atoms: number[]        // atom serial indices
  secondaryStructure?: 'helix' | 'sheet' | 'loop'
}

interface Chain {
  id: string
  residues: Residue[]
}

interface MoleculeData {
  atoms: Atom[]
  bonds: Bond[]
  chains: Chain[]
  center: [number, number, number]
  boundingRadius: number
}

5. Parsers

PDBParser.ts

Parse ATOM, HETATM, CONECT, HELIX, and SHEET records. Return MoleculeData. Must handle multi-model files (use MODEL 1 only). Must not throw on missing optional fields.

export function parsePDB(text: string): MoleculeData

mmCIFParser.ts

Parse _atom_site loop. Map label_atom_id, label_comp_id, label_asym_id, label_seq_id, Cartn_x/y/z. Return same MoleculeData shape.

export function parseMMCIF(text: string): MoleculeData

RCSB fetch

Fetch mmCIF by accession code using https://files.rcsb.org/download/{id}.cif. Validate 4-char alphanumeric accession before fetch. Handle 404 with a user-facing error message.

6. Web Worker (engine)

Heavy computation runs in moleculeWorker.ts, imported via Vite's new Worker(new URL('./engine/moleculeWorker.ts', import.meta.url), { type: 'module' }).

The worker accepts a MoleculeTask message and posts back a MoleculeResult:

type MoleculeTask =
  | { type: 'process'; atoms: Atom[] }

type MoleculeResult =
  | { type: 'bonds'; bonds: Bond[] }
  | { type: 'secondaryStructure'; residues: Residue[] }
  | { type: 'spatialIndex'; ready: true }

BondCalculator.ts

Calculate covalent bonds by element-pair distance thresholds. Use a k-d tree (SpatialIndex.ts) to avoid O(n²) search. Do not recalculate if CONECT records are present in PDB input.

StructureAnalyser.ts

Assign secondary structure from HELIX/SHEET records if present, otherwise use a simplified DSSP heuristic (backbone dihedral angles). Annotate each Residue with secondaryStructure.

7. Rendering

MoleculeViewer.ts — scene orchestrator

Owns the Three.js Scene, Camera, Renderer, OrbitControls, and CSS2DRenderer for labels. Exposes a clean public API consumed by ViewerCanvas.tsx:

class MoleculeViewer {
  constructor(canvas: HTMLCanvasElement)
  load(data: MoleculeData): void
  setRepresentation(mode: RepresentationMode): void
  setColorScheme(scheme: ColorScheme): void
  highlightResidue(seq: number, chainID: string): void
  clearHighlights(): void
  addMeasurement(atomA: Atom, atomB: Atom): void
  clearMeasurements(): void
  takeScreenshot(): string           // returns data URL
  resetCamera(): void
  dispose(): void
}

Representation modes

ModeImplementationPerformance note
ball-and-stickInstancedMesh per element for atoms; cylinder Mesh per bondDefault mode
space-fillingInstancedMesh per element; van der Waals radii from constantsNo bonds rendered
ribbonCatmullRomCurve3 through Cα atoms; TubeGeometry coloured by secondary structureFastest for large proteins
backboneBufferGeometry line through N, Cα, C atomsMinimal geometry

LOD strategy

Apply Three.js LOD: full geometry at 0–40 Å camera distance, simplified (reduced sphere segments, bonds hidden) at 40–120 Å, backbone-only beyond 120 Å.

WebGPU progressive enhancement

const renderer = await (async () => {
  if ('gpu' in navigator) {
    try {
      const { WebGPURenderer } = await import('three/webgpu')
      const r = new WebGPURenderer({ canvas })
      await r.init()
      return r
    } catch {}
  }
  return new THREE.WebGLRenderer({ canvas, antialias: true })
})()

9. Components

ViewerCanvas.tsx

Mounts MoleculeViewer against a canvas ref. Subscribes to store; calls viewer API on state changes. Handles resize via ResizeObserver. Cleans up on unmount via viewer.dispose().

ControlsPanel.tsx

Renders representation toggle (4 buttons), colour scheme selector, hydrogen toggle, chain visibility checkboxes, and screenshot button. All actions dispatch to Zustand store.

FileUpload.tsx

Drag-and-drop zone accepting .pdb and .cif files. Text input for RCSB accession code with fetch button. On load, dispatches to moleculeSlice and triggers worker processing.

SelectionInfo.tsx

Displays element, residue name, chain, sequence number, and coordinates for the selected atom. Empty state when nothing selected.

MeasureOverlay.tsx

Lists active distance measurements as a stack of pills showing atom pair and distance in Å. Includes a clear-all button.

12. Requirements

MUST

Parse valid PDB and mmCIF files without throwing. Surface parse errors as user-visible messages, not console errors.

MUST

Run bond calculation and secondary structure detection in a Web Worker. The main thread must not block during processing.

MUST

Maintain 60 fps during rotation for proteins up to 10,000 atoms in ball-and-stick mode on a mid-range desktop (tested in Chrome).

MUST

Support touch interactions on iOS Safari and Android Chrome (rotate, pan, zoom, tap-to-select).

MUST

All TypeScript compiled with strict: true. No any types except at parser input boundaries.

SHOULD

Use WebGPU renderer when available; fall back to WebGL2 silently.

SHOULD

Support RCSB accession fetch as an alternative to file upload.

SHOULD

Be installable as a PWA (manifest + service worker caching for the app shell).

MUST NOT

Block the main thread for more than 16 ms during any user interaction.

MUST NOT

Make network requests except for RCSB fetch (user-initiated) and CDN assets at build time.

14. Out of scope (v1)

Molecular dynamics trajectory playback · electron density map overlay · solvent-accessible surface computation · multi-structure alignment · structure editing or export · user authentication or persistence.

Intended for use as a coding agent prompt. Implement sections in order: types → parsers → engine → renderers → store → components.
